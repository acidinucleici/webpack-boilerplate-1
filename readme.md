# webpack.config.js
Config for assets, es6, and css-sass-post-css loading.

## Building
* Production build: npm run build
* Development build: npm run build:dev
* Webpack Dev Server: npm run dev

## Directory structure
```
├── src/
│   ├── sass/
│   ├── js/
│   ├── images/
│   ├── fonts/   
│   ├── entry.js
├── index.html   
├── webpack.config.js/
```
